Documentação.

Esse é um pequeno projeto de API REST, na qual retorna frases, esse projeto foi todo construido usando python, onde consiste em um webserver construido em Flask na qual realiza a leitura de um arquivo JSON, e através de um recurso nativo do python o random, sorteia uma frase aleatoria para retornar via JSON.
Os tipos de frases disponiveis:

  • Frases motivacionais.

Como funciona ?

O funcionamento da API é simples, basta realizar um request do tipo GET para a API por meio da rota desejada, após isso você vai receber a frase no formato JSON.
A cada novo request enviado para API ela vai sempre retonar uma frase diferente, essa API pde ser implementada em diversos tipos de sistemas para cobrir espaços vazios ou em telas de loading.
